# 左侧菜单UI更新说明

## 更新内容

根据设计图重新实现了左侧底部菜单的UI，使其更符合网易云音乐的设计风格。

## 实现的功能

### 1. 菜单项样式

#### 精选
- 图标：🎵 音符
- 普通样式，可选中高亮

#### 播客
- 图标：📻 广播
- 普通样式，可选中高亮

#### 关注（特殊样式）
- 图标：💬 对话气泡
- 红色背景（#ec4141）
- 白色文字
- 圆角矩形（8px）
- 始终保持红色背景，突出显示

#### 我喜欢的音乐
- 图标：❤ 心形（灰色）
- 右侧带波浪图标 ♡
- 可选中高亮（背景变深色）

#### 最近播放
- 图标：圆形背景（灰色）+ 字母"L"
- 可选中高亮

#### 下载管理
- 图标：方形背景（灰色）+ 下载箭头"↓"
- 可选中高亮

### 2. 创建的歌单
- 文件夹图标 📁
- 显示"创建的歌单 0"
- 右侧带"+"按钮
- 灰色文字
- 可点击创建新歌单

## 样式规范

### 颜色
- 背景色：#1a1a21（深色）
- 选中背景：#2d2d37（深灰色）
- 关注按钮：#ec4141（红色）
- 普通文字：#cccccc（浅灰色）
- 选中文字：#ffffff（白色）
- 图标背景：#666666（中灰色）
- 次要文字：#999999（灰色）

### 尺寸
- 菜单项高度：48px
- 圆角：8px
- 图标大小：20-24px
- 字体大小：15px（主要）、14px（次要）
- 左右边距：16px
- 图标文字间距：12px

### 交互
- 鼠标悬停：显示手型光标
- 点击反馈：切换选中状态
- 选中效果：背景色变化 + 文字颜色变化

## 与之前的区别

### 移除的内容
- 移除了 MenuItem 组件的使用
- 移除了 Divider 分割线
- 移除了 SectionHeader 组件
- 移除了"收藏的歌单"部分

### 新增的内容
- 直接使用 Rectangle 实现自定义样式
- 关注按钮的特殊红色背景样式
- 图标背景（圆形/方形）
- 更精细的间距控制

### 优化的内容
- 更符合设计图的视觉效果
- 更清晰的层次结构
- 更好的交互反馈

## 代码结构

```qml
Rectangle {
    Flickable {
        Column {
            // 精选
            MenuItem { ... }
            
            // 播客
            MenuItem { ... }
            
            // 关注（红色背景）
            Rectangle {
                color: "#ec4141"
                Row {
                    Text { text: "💬" }
                    Text { text: "关注" }
                }
            }
            
            // 我喜欢的音乐
            Rectangle {
                Row {
                    Text { text: "❤" }
                    Text { text: "我喜欢的音乐" }
                    Text { text: "♡" }  // 波浪图标
                }
            }
            
            // 最近播放
            Rectangle {
                Row {
                    Rectangle {  // 圆形背景
                        Text { text: "L" }
                    }
                    Text { text: "最近播放" }
                }
            }
            
            // 下载管理
            Rectangle {
                Row {
                    Rectangle {  // 方形背景
                        Text { text: "↓" }
                    }
                    Text { text: "下载管理" }
                }
            }
            
            // 创建的歌单
            Rectangle {
                Row {
                    Text { text: "📁" }
                    Text { text: "创建的歌单 0" }
                    Text { text: "+" }
                }
            }
        }
    }
}
```

## 使用方法

1. 启动应用
2. 查看左侧菜单
3. 点击不同菜单项查看选中效果
4. 点击"+"按钮创建歌单

## 技术要点

1. **自定义样式**：使用 Rectangle 实现完全自定义的菜单项样式
2. **图标处理**：使用 Emoji 图标 + 背景实现不同的图标效果
3. **状态管理**：使用 selectedIndex 管理选中状态
4. **布局控制**：使用 Row + spacing 实现精确的布局
5. **交互反馈**：使用 MouseArea 实现点击和悬停效果
