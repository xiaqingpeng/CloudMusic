## [1.0.0] - 2026-04-16

### Added
- Initial release
- Music player with play/pause/next/previous controls
- Playlist management with drawer interface
- Audiobook support
- Search functionality
- Official playlist carousel
- Dark theme UI
- Responsive layout
- macOS DMG packaging support

### Features
- **Left Panel**
  - Navigation menu with hover effects
  - My favorite music
  - Recent plays
  - Download manager
  - Created playlists
  - Collected playlists

- **Right Panel**
  - Search bar with suggestions
  - Hot search popup
  - Login functionality
  - Banner swiper
  - Official playlists with horizontal scroll
  - Audiobook grid view

- **Bottom Panel**
  - Album cover display
  - Song information
  - Playback controls
  - Progress bar
  - Playlist drawer with 21 songs

### Technical
- Qt 6.2+ support
- QML-based UI
- CMake build system
- ViewModel pattern for data management
- Modular component architecture

## [0.1.0] - 2026-01-01

### Added
- Project initialization
- Basic UI structure
- Core components
