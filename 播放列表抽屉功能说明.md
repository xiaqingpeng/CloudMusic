# 播放列表抽屉功能实现说明

## 功能概述

实现了一个从右侧弹出的播放列表抽屉，点击底部播放控制栏的"☰"按钮即可打开。

## 实现的功能

### 1. 抽屉动画
- 从右侧平滑滑入/滑出
- 带有半透明背景遮罩
- 点击遮罩区域可关闭抽屉
- 动画时长300ms，使用缓动曲线

### 2. 播放列表头部
- 显示"播放列表"标题和歌曲数量（动态显示，当前21首）
- "收藏全部"按钮（带⊕图标）
- "清空"按钮（带🗑图标）

### 3. 推荐开关
- 显示推荐提示："在列表下方推荐你喜欢的相似歌曲"
- 带有开关按钮（可点击切换）

### 4. 播放列表（支持滚动）
- 显示21首歌曲（可滚动查看）
- 自定义滚动条样式（8px宽，圆角，悬停变色）
- 每首歌曲包含：
  - 专辑封面（50x50）
  - 歌曲名称
  - MV标签（如果有视频）
  - 音质标签（超清母带/Hi-Res）
  - 歌手名称
  - 时长
  - 操作按钮（分享🔗、喜欢♡、添加⊕、更多⋯）
- 当前播放歌曲高亮显示（背景色+红色文字）
- 当前播放歌曲封面显示播放图标▶
- 鼠标悬停效果（非当前播放项显示浅灰背景）
- 点击歌曲可切换播放

## 核心代码说明

### 1. PlaylistDrawer.qml
```qml
Item {
    id: drawer
    visible: false
    property bool opened: false
    property var playlist: [...]  // 21首歌曲数据
    
    // 半透明遮罩
    Rectangle {
        color: "#80000000"
        opacity: drawer.opened ? 1 : 0
        MouseArea {
            onClicked: drawer.close()
        }
    }
    
    // 抽屉主体
    Rectangle {
        width: 420
        anchors.rightMargin: drawer.opened ? 0 : -width
        Behavior on anchors.rightMargin {
            NumberAnimation { duration: 300; easing.type: Easing.OutCubic }
        }
        
        // 播放列表 ListView
        ListView {
            clip: true
            boundsBehavior: Flickable.StopAtBounds
            
            // 自定义滚动条
            ScrollBar.vertical: ScrollBar {
                width: 8
                contentItem: Rectangle {
                    radius: 4
                    color: parent.pressed ? "#999999" : "#cccccc"
                }
            }
            
            delegate: Rectangle {
                // 鼠标悬停效果
                property bool hovered: false
                MouseArea {
                    hoverEnabled: true
                    onEntered: parent.hovered = true
                    onExited: parent.hovered = false
                }
            }
        }
    }
    
    function open() { ... }
    function close() { ... }
}
```

### 2. MyBottomRect.qml
```qml
Rectangle {
    signal playlistRequested()  // 播放列表请求信号
    
    ActionButton {
        icon: "☰"
        onClicked: bottomRect.playlistRequested()
    }
}
```

### 3. Main.qml
```qml
MyWindowRect {
    MyBottomRect {
        onPlaylistRequested: playlistDrawer.open()
    }
    
    PlaylistDrawer {
        id: playlistDrawer
        z: 1000  // 确保在最上层
    }
}
```

## 使用方法

1. 启动应用
2. 点击底部播放控制栏右侧的"☰"按钮
3. 播放列表抽屉从右侧滑入
4. 点击遮罩区域或按ESC键关闭抽屉

## 样式说明

- 抽屉宽度：420px
- 背景色：#f5f5f7（浅灰色）
- 头部背景：#ffffff（白色）
- 当前播放项背景：#e8e8ea（浅灰色）
- 高亮颜色：#ec4141（红色）
- 音质标签：红色边框+浅红背景

## 技术要点

1. **信号机制**：使用Qt信号在组件间通信
2. **动画**：使用Behavior实现平滑动画
3. **层级管理**：使用z-index确保抽屉在最上层
4. **布局**：使用Row/Column进行灵活布局
5. **状态管理**：使用property管理抽屉开关状态
