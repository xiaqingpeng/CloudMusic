## 技术要点

### Qt Multimedia 关键 API

```qml
// 播放控制
audioPlayer.play()      // 开始播放
audioPlayer.pause()     // 暂停
audioPlayer.stop()      // 停止

// 进度控制
audioPlayer.position    // 当前位置（毫秒）
audioPlayer.duration    // 总时长（毫秒）

// 音量控制
audioOutput.volume      // 音量 (0.0 - 1.0)
audioOutput.muted       // 静音状态

// 播放状态
audioPlayer.playbackState  // PlayingState, PausedState, StoppedState
audioPlayer.mediaStatus    // LoadingMedia, LoadedMedia, EndOfMedia, etc.
```

### 资源路径规则

```
文件系统路径: src/resources/5156edu-2054-6162.mp3
QRC 前缀:     /audio
QML 访问路径: qrc:/audio/resources/5156edu-2054-6162.mp3
```
