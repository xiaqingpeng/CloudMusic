## 快速查询：我该重点学什么？

### 🎯 核心必备（每天都用）
1. **Qt/QML 开发**：QObject、信号槽、QML 语法、属性绑定、C++/QML 集成
2. **Linux 基础**：进程管理、文件系统、系统调用、交叉编译
3. **车载通信**：CAN 总线、SocketCAN、SOME/IP、DBC 文件
4. **图形渲染**：OpenGL ES、Qt Scene Graph、Wayland、DRM/KMS
5. **多媒体**：GStreamer、ALSA、硬件加速
6. **硬件接口**：GPIO、I2C、LVDS、触摸屏、Ethernet
7. **开发工具**：C++、Qt Creator、CMake、GDB、Git

### 🔥 高频使用（每周都用）
1. **高级 Qt**：多线程、网络编程、数据库、性能优化
2. **驱动开发**：字符设备、I2C/SPI 驱动、中断处理、设备树
3. **AUTOSAR**：Adaptive AUTOSAR、ara::com
4. **车载诊断**：UDS 协议、DTC 管理、刷写流程
5. **音视频**：FFmpeg、V4L2、硬件编解码
6. **系统优化**：实时性能、启动优化、内存优化

### 💼 工程实践（项目必备）
1. **构建系统**：Yocto/BitBake、Recipe 编写、BSP 开发
2. **功能安全**：ISO 26262、MISRA C++、静态分析、单元测试
3. **网络安全**：ISO 21434、安全启动、OTA 安全
4. **项目管理**：CI/CD、Agile、代码审查
